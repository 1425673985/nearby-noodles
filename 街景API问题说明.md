# 街景API 404错误说明

## 问题

访问 `https://apis.map.qq.com/ws/streetview/v1/image` 返回404错误，说明该API路径不存在或不正确。

## 可能的原因

1. **腾讯地图可能没有公开的街景静态图API**
   - 腾讯地图的街景服务可能只通过SDK提供，不提供直接的HTTP API

2. **API路径不正确**
   - 可能使用了错误的API路径
   - 需要查看腾讯地图官方文档确认正确的接口

3. **需要特定的权限或服务**
   - 街景服务可能需要特殊的API Key权限
   - 或者需要购买特定的服务套餐

## 建议的解决方案

### 方案1：使用腾讯地图小程序SDK（推荐）
如果腾讯地图提供小程序SDK支持街景，使用SDK方式调用，而不是直接调用HTTP API。

### 方案2：使用静态地图API替代
如果街景API不可用，可以暂时使用腾讯地图的静态地图API，显示面馆位置的地图：
- API路径：`https://apis.map.qq.com/ws/staticmap/v2/`
- 这个接口应该是可用的

### 方案3：使用其他服务
考虑使用其他提供街景服务的API，但需要确保兼容性和合规性。

## 当前代码状态

代码中使用的是：
```
https://apis.map.qq.com/ws/streetview/v1/image
```

这个路径可能不存在，需要确认正确的API路径或使用替代方案。
