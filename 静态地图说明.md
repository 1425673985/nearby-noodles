# 静态地图功能说明

## 已完成的功能

1. ✅ **移除了上传店铺照片功能**
   - 删除了所有上传照片相关的代码
   - 移除了上传按钮和相关样式
   - 移除了相机权限配置

2. ✅ **集成了腾讯地图静态图API**
   - 添加了 `generateStaticMapUrl()` 函数
   - 自动生成以面馆位置为中心的静态地图图片
   - 地图上标注了面馆位置（橙色标记点）

## 静态地图参数

- **中心点**：面馆的经纬度坐标
- **缩放级别**：16（10-20之间，数值越大越详细）
- **图片尺寸**：750*500像素（适配小程序显示）
- **标记点**：大图标、橙色(#FF6B35)、标签A

## 地图URL格式

```
https://apis.map.qq.com/ws/staticmap/v2/?center=纬度,经度&zoom=16&size=750*500&markers=size:large|color:0xFF6B35|label:A|纬度,经度&key=API_KEY
```

## 注意事项

1. **域名配置**：确保在微信公众平台配置了 `https://apis.map.qq.com` 到 request合法域名
2. **API Key**：使用的是配置在 `utils/config.js` 中的腾讯地图API Key
3. **图片加载**：如果地图图片加载失败，会显示"加载地图中..."的占位符

## 功能特点

- 自动显示：每次加载或切换面馆时，自动生成并显示对应的静态地图
- 位置准确：地图以面馆位置为中心，并标注了具体位置
- 样式统一：标记点颜色与小程序主题色一致（橙色）

## 如需调整

如果需要调整地图参数，可以修改 `generateStaticMapUrl()` 函数中的参数：
- `zoom`：调整缩放级别（建议10-20）
- `size`：调整图片尺寸（最大2048*2048）
- `markers`：调整标记点样式、颜色、标签等
