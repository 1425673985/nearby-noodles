# 模拟数据说明

## 当前状态

代码已切换为使用**模拟数据模式**，不再调用真实的高德地图API。这样可以先测试界面效果，无需配置API Key。

## 模拟数据内容

目前代码中包含5个模拟面馆数据：

1. **老北京炸酱面** - 评分4.8分，256条评价
2. **四川担担面馆** - 评分4.6分，189条评价  
3. **兰州牛肉拉面** - 评分4.7分，312条评价
4. **陕西油泼面** - 评分4.5分，142条评价
5. **重庆小面** - 评分4.9分，428条评价

每个面馆都包含：
- 店名
- 地址
- 评分
- 评价数量
- 距离（基于用户位置计算）
- 电话
- 用户评价（根据评分自动生成）

## 切换回真实API

当你准备好使用真实的高德地图API时，可以按照以下步骤切换：

1. **打开文件** `pages/index/index.js`

2. **找到 `searchNearbyRestaurants` 函数**（大约在第177行）

3. **注释掉模拟数据部分**（第182-201行）：
   ```javascript
   // ===== 模拟数据方案（当前使用） =====
   // setTimeout(() => {
   //   const mockRestaurants = this.generateMockRestaurants(location)
   //   ... (注释掉整个setTimeout部分)
   // }, 500)
   ```

4. **取消注释真实API部分**（第203-227行）：
   ```javascript
   // ===== 真实API方案（暂时注释） =====
   apiUtils.searchNearbyRestaurants(location.latitude, location.longitude, config.SEARCH_RADIUS, config.SEARCH_KEYWORD)
     .then((restaurants) => {
       ... (取消注释整个API调用部分)
     })
   ```

5. **配置API Key**
   - 在 `utils/config.js` 中配置你的高德地图API Key
   - 在微信公众平台配置request合法域名

6. **重新编译**小程序

## 注意事项

- 模拟数据会基于用户的实际位置计算距离
- 模拟数据的位置是在用户位置附近随机偏移生成的
- 图片使用的是默认占位符，如需显示真实图片需要准备图片资源
- 切换回真实API后，数据将来自高德地图的实际POI数据
